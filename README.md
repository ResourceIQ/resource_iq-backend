## Getting Started

### Prerequisites

- Python 3.10 or higher
- PostgreSQL database

### Installation

1. **Clone the repository**

```bash
git clone <your-repository-url>
cd resource_iq-backend
```

2. **Set up Python virtual environment**

```bash
python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate
```

3. **Install dependencies**

You can use either `pip` or `uv`:

```bash
# Using pip
pip install -e .

# Or using uv (faster)
uv sync
```

```

4. **Configure environment variables**

Create a `.env` file in the root directory with the following:

```env
# Backend
DOMAIN=localhost
ENVIRONMENT=local
BACKEND_CORS_ORIGINS=["http://localhost","http://localhost:8000"]
SECRET_KEY=changethis
PROJECT_NAME="FastAPI Backend"
POSTGRES_SERVER=localhost
POSTGRES_PORT=5432
POSTGRES_DB=app
POSTGRES_USER=postgres
POSTGRES_PASSWORD=changethis
FIRST_SUPERUSER=admin@example.com
FIRST_SUPERUSER_PASSWORD=changethis
```

**Important:** Change `changethis` values to secure passwords. Generate secret keys with:

```bash
python -c "import secrets; print(secrets.token_urlsafe(32))"
```

5. **Set up PostgreSQL database**

Make sure PostgreSQL is running and create a database:

```sql
CREATE DATABASE app;
```

6. **Run database migrations**

```bash
alembic upgrade head
```

7. **Create initial data (optional)**

```bash
python -m app.initial_data
```

8. **Run the application**

```bash
uvicorn app.main:app --reload
uv run fastapi dev app/main.py
```

The API will be available at `http://localhost:8000`

### Interactive API Documentation

Once the application is running, you can access:

- **Swagger UI**: http://localhost:8000/docs
- **ReDoc**: http://localhost:8000/redoc

## Testing

Run tests with:

```bash
pytest
```
